## 🌐 ACCEDER DESDE OTRO DISPOSITIVO

Para acceder desde otra computadora en la red local:

1. En la máquina con el servidor, ejecuta:
```powershell
ipconfig
```
Busca "IPv4 Address" (ej: 192.168.1.100)

2. Desde otra máquina, abre:
```
http://192.168.1.100:5000
```

3. Para acceder desde Internet necesitarías:
   - Configurar port forwarding en router
   - Usar un servicio como ngrok

## 🆘 SOLUCIÓN DE PROBLEMAS

### ❌ "No se encuentra 'run.bat'" o "No funciona doble clic"

**Solución:**
```powershell
cd "c:\Users\samue\Documents\Contador"
.\run.ps1
```

---

### ❌ Error: "Module not found: flask"

**Solución:**
```powershell
pip install flask python-docx
```

---

### ❌ "Port 5000 is already in use"

**Solución Rápida:**
- Espera 30 segundos y reinicia
- O cierra otras aplicaciones Flask

**Solución Permanente:**
- Edita `app.py`
- Última línea: cambia `port=5000` a `port=5001`
- Guarda y ejecuta nuevamente

---

### ❌ "La interfaz se ve mal"

**Solución:**
- Actualiza el navegador: `Ctrl+F5`
- Usa un navegador moderno (Chrome, Edge)
- Limpia caché: `Ctrl+Shift+Del`

---

### ❌ "No puedo cargar archivos .docx"

**Solución:**
```powershell
pip install python-docx
```

---

### ❌ PowerShell: "Cannot be loaded because running scripts is disabled"

**Solución:**
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
Presiona `Y` y Enter cuando te pida confirmación.
