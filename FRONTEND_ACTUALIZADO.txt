🔄 LÓGICA DE CÁLCULO DEL PROGRESO:
═══════════════════════════════════════════════════════════════════════════════════

Basado en: 
  - data.stats (número de stats obtenidos)
  - data.lines_for_next_stat (líneas faltantes)
  - increments = [80, 40, 30] (ciclo de incrementos)

Fórmula:
  1. currentCycleIndex = data.stats % 3
     → Posición en el ciclo (0, 1, o 2)
  
  2. currentIncrement = increments[currentCycleIndex]
     → Cuántas líneas se necesitan para este stat
  
  3. progress = ((currentIncrement - lines_for_next_stat) / currentIncrement) * 100
     → Porcentaje de progreso en el stat actual

Ejemplo:
  - Stats: 5 (Stat 5)
  - Lines_for_next_stat: 15
  - currentCycleIndex = 5 % 3 = 2
  - currentIncrement = increments[2] = 30
  - progress = ((30 - 15) / 30) * 100 = 50%

// Nuevos incrementos del ciclo
const increments = [80, 40, 30];

// Determinar qué incremento se está cumpliendo
const currentCycleIndex = data.stats % 3;
const currentIncrement = increments[currentCycleIndex];

// Calcular progreso en porcentaje
progress = ((currentIncrement - data.lines_for_next_stat) / currentIncrement) * 100;

// Asegurar que el progreso esté entre 0 y 100%
progress = Math.max(0, Math.min(100, progress));
