═══════════════════════════════════════════════════════════════════════════════════
                           FUNCIONALIDADES PRINCIPALES
═══════════════════════════════════════════════════════════════════════════════════

✓ LIMPIEZA AUTOMÁTICA:
  • Etiquetas BBCode [b], [i], [color], etc.
  • Etiquetas HTML <div>, <span>, <p>, etc.
  • Normalización de espacios y saltos de línea

✓ ANÁLISIS DE TEXTO:
  • Conteo de caracteres (solo texto plano)
  • Cálculo de líneas (caracteres ÷ 100)
  • Evaluación de nivel de stat obtenido
  • Cálculo de líneas faltantes para próximo stat

✓ PROGRESIÓN DE STATS (Cíclica):
  80 líneas   → +1 stat
  200 líneas  → +2 stats
  350 líneas  → +3 stats
  430 líneas  → +4 stats ← Ciclo reinicia
  ...y así sucesivamente

✓ INTERFAZ DE USUARIO:
  • Diseño moderno y responsivo
  • Drag & drop para archivos
  • Visualización en tiempo real
  • Barra de progreso animada
  • Indicadores visuales claros

✓ SEGURIDAD:
  • Validación de archivos
  • Límite de tamaño (16MB)
  • Sanitización de nombres
  • Limpieza automática de temporales

                              USO DEL SISTEMA
═══════════════════════════════════════════════════════════════════════════════════

FLUJO DE USO:

1. Usuario abre http://127.0.0.1:5000
   ↓
2. Ve interfaz moderna con área de upload
   ↓
3. Arrastra o selecciona archivo (.txt o .docx)
   ↓
4. Sistema procesa instantáneamente:
   - Extrae texto
   - Limpia etiquetas BBCode y HTML
   - Cuenta caracteres
   - Calcula líneas
   - Calcula stats
   - Determina líneas faltantes
   ↓
5. Muestra resultados en tiempo real:
   - Stats obtenidos
   - Total de líneas
   - Total de caracteres
   - Líneas faltantes para próximo stat
   - Barra de progreso visual
   ↓
6. Usuario puede subir otro archivo o exportar si lo desea

❓ PROBLEMAS COMUNES:

P: "No se ejecuta run.bat"
R: Usa run.ps1 en PowerShell en su lugar

P: "Port 5000 is already in use"
R: Edita app.py y cambia port=5001

P: "No se cargan archivos .docx"
R: Reinstala: pip install python-docx

P: "La interfaz se ve mal"
R: Actualiza navegador (Ctrl+F5)

❓ PREGUNTAS FRECUENTES:

P: ¿Puedo usar esto en producción?
R: No, es una herramienta local. Para producción, usa un servidor real.

P: ¿Puedo modificar la progresión de stats?
R: Sí, edita la variable 'increments' en app.py

P: ¿Puedo cambiar los colores?
R: Sí, edita templates/index.html y busca clases Tailwind

P: ¿Es seguro?
R: Sí, pero solo para uso local. No expon a Internet sin auth.
